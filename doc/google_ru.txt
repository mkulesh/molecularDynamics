ММД - это реализация классического метода молекулярной динамики для Андроида. Метод молекулярной динамики – это метод компьютерного моделирования движения частиц (молекул, атомов). Моделирование осуществляется для ансамбля частиц при заданных начальных и граничных условиях, а также с использованием функции потенциала, описывающего взаимодействие частиц.

Траектории частиц определяются численно, с использованием известного и хорошо зарекомендовавшего себя скоростного метода Верле для интегрирования уравнений движения Ньютона. Парное взаимодействие частиц описывается аналитически и задается потенциальной функцией, значение которой зависит от расстояния между двумя частицами.

В физике, ММД используется для изучения динамических эффектов на микроуровне, которые на данном этапе развития науки пока еще нельзя наблюдать непосредственно. В контексте мобильных устройств, ММД может использоваться как небольшая мобильная лаборатория, помогающая понять основы физики микрочастиц. Более того, этот метод позволяет генерировать интересные динамические картинки, которые легко использовать в качестве живых обоев.

Основные функции приложения:

- Приложение может быть выбрано в настройках экрана в качества живых обоев, которые показывают “жизнь” сконфигурированной системы частиц.
- В режиме живых обоев приложение может использовать системные статические обои в качестве фонового изображения. Это позволяет комбинировать Ваши любимые картинки с ансамблем частиц, “плывущих” сверху
- Изменения настроек, сделанные в основном приложении, сразу перенимаются живыми обоями, что позволяет легко эти самые обои настраивать
- Возможность задавать начальную структуру ансамбля частиц использую прямоугольную  или диагональную сетки
- Возможность выбора граничных условий по нормировке энергии системы (постоянство полной энергии, постоянство кинетической энергии, или без ограничений)
- В случае фиксирования кинетической энергии, имеется возможность менять термальный коэффициент, который эмитирует нагрев или охлаждение системы
- Возможность изменения силы тяжести. Направление силы тяжести можно менять поворотом устройства
- Возможность изменения и визуализации выбранного потенциала парного взаимодействия. Реализованы следующие потенциалы: Леннарда-Джонса, Морзе и Борна-Майера
- Возможность выбора картинки для частицы из небольшой встроенной библиотеки картинок
- Возможность приближения и перемещения по выбранной физической области

Примечание: даже в режиме живых обоев, это приложение выполняет большое количество  вычислений для решения системы уравнений движения частиц. Поэтому, предназначено оно в первую очередь для устройств с многоядерным процессором и мощным аккумулятором.
